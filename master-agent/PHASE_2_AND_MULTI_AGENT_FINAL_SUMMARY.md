#### Orchestration Engine (450 lines)

**Execution Strategies:**

**1. Best Agent (Single)**
- Routes to single best agent
- Based on capability, health, priority
- Cost: 1x, Speed: Fastest
- Use: Standard tasks, cost optimization

**2. Parallel Execution**
- Executes on ALL capable agents
- Returns consensus result
- Cost: 3x, Speed: Similar to single
- Use: Critical decisions, multiple perspectives

**3. Automatic Fallback**
- Tries agents sequentially (max 3)
- Auto-retry on failure
- Cost: 1x-3x, Reliability: 99.9%+
- Use: Reliability priority

**4. Sequential Pipeline**
- Each output â†’ next input
- Full context passing
- Cost: Nx (N tasks)
- Use: Multi-step workflows
